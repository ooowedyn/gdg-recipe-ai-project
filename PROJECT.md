# 프로젝트 기획서 및 구현 기획서

## 1. 프로젝트명

사진 기반 식재료 인식 및 한식 레시피 추천 앱

## 2. 프로젝트 배경

냉장고에 식재료가 있어도 어떤 요리를 만들 수 있는지 바로 떠올리기 어려운 경우가 많다. 특히 요리 경험이 적은 사용자는 재료를 가지고 있어도 적절한 메뉴를 찾기 위해 직접 검색해야 하는 불편함이 있다.

## 3. 프로젝트 목적

카메라로 식재료를 촬영하면 사진 속 재료를 자동으로 인식하고, 인식된 재료만으로 만들 수 있는 한식 메뉴를 추천하는 앱을 제작한다.

이를 통해 사용자가 직접 검색어를 입력하지 않아도 현재 가진 재료를 기준으로 요리 방법을 쉽게 찾을 수 있도록 한다.

## 4. 주요 사용자

요리 경험이 적고, 집에 있는 재료로 무엇을 만들지 고민하는 사용자

예시 사용자:

- 자취생
- 신혼부부
- 요리 초보자
- 냉장고 속 남은 재료를 활용하고 싶은 사람

## 5. 핵심 기능

### 5-1. 사진 속 식재료 인식

사용자가 식재료 사진을 촬영하면 AI가 사진 속 주요 식재료를 인식한다.

### 5-2. 식재료 리스트 생성

인식된 식재료를 리스트 형태로 보여준다.
사용자는 잘못 인식된 재료를 수정하거나 누락된 재료를 직접 추가할 수 있다.

### 5-3. 보유 재료 기반 메뉴 추천

식재료 리스트에 있는 재료만으로 만들 수 있는 한식 메뉴를 추천한다.

### 5-4. 레시피 단계별 시각화

추천된 메뉴를 선택하면 필요한 재료, 조리 순서, 조리 시간, 난이도 등을 단계별로 보여준다.

## 6. MVP

MVP 단계에서는 다음 기능을 중심으로 개발한다.

1. 사진 속 식재료 인식
2. 인식된 식재료 리스트 생성
3. 보유 재료 기반 한식 메뉴 추천
4. 레시피 단계별 시각화

## 7. 차별점

기존 레시피 검색 서비스는 사용자가 직접 재료명을 입력해 검색해야 한다. 하지만 본 프로젝트는 사진 속 식재료를 자동으로 인식하여 보유 재료 리스트를 생성한다.

또한 단순히 레시피를 보여주는 것이 아니라, 현재 가진 재료만으로 만들 수 있는 메뉴를 추천한다는 점에서 차별성을 가진다.

## 8. 구현 목표

사용자가 식재료 사진을 촬영하면 AI가 사진 속 주요 식재료를 인식하고, 이를 리스트 형태로 정리한다. 이후 생성된 식재료 리스트를 바탕으로 현재 가진 재료만으로 만들 수 있는 한식 메뉴를 추천하고, 선택한 메뉴의 조리 과정을 단계별로 보여주는 앱을 구현한다.

## 9. 구현 범위

이번 MVP에서는 다음 기능을 구현한다.

1. 식재료 사진 입력 기능
2. 사진 속 식재료 인식 기능
3. 인식된 식재료 리스트 생성 기능
4. 식재료 수정 및 추가 기능
5. 보유 재료 기반 한식 메뉴 추천 기능
6. 레시피 단계별 시각화 기능

보유 양념, 조리 도구, 사용자 취향 반영, 냉장고 실시간 스캔 기능은 MVP 이후 확장 기능으로 분리한다.

## 10. 개발 순서

1. 사용자 흐름 정리
2. 화면 구조 설계
3. 식재료 리스트 데이터 형식 정리
4. 레시피 추천 데이터 형식 정리
5. 이미지 입력 기능 구현
6. 식재료 인식 AI 연결
7. 인식 결과 리스트화 기능 구현
8. 식재료 추가, 수정, 삭제 기능 구현
9. 재료 기반 메뉴 추천 기능 구현
10. 레시피 단계별 시각화 화면 구현
11. 전체 흐름 테스트 및 오류 수정

## 11. 기대 효과

사용자는 집에 있는 식재료만으로 만들 수 있는 요리를 쉽게 찾을 수 있다. 특히 요리 초보자도 복잡한 검색 과정 없이 재료 인식 결과를 바탕으로 적절한 한식 메뉴와 조리 과정을 확인할 수 있다.

## 12. 향후 개선 방향

MVP 이후에는 다음 기능을 추가로 고려한다.

- 냉장고 실시간 스캔 기능
- 보유 양념 및 조리 도구 반영
- 사용자 취향 기반 추천
- 조리 시간, 난이도, 매운맛 여부 필터
- 부족한 재료 대체 추천
