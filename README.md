# 사진 기반 식재료 인식 및 한식 레시피 추천 앱

## 프로젝트 소개

이 프로젝트는 사용자가 식재료 사진을 촬영하면 AI가 사진 속 식재료를 인식하고, 인식된 재료만으로 만들 수 있는 한식 메뉴를 추천하는 앱을 제작하는 것을 목표로 한다.

요리 경험이 적은 사용자가 직접 검색어를 입력하지 않아도, 현재 가지고 있는 재료를 기준으로 만들 수 있는 요리와 조리 과정을 쉽게 확인할 수 있도록 한다.

## 핵심 MVP

이번 프로젝트의 MVP는 다음 4가지 기능을 중심으로 한다.

1. 사진 속 식재료 인식
2. 인식된 식재료 리스트 생성
3. 해당 재료로 만들 수 있는 한식 메뉴 추천
4. 레시피 단계별 시각화

## 프로젝트 흐름

사진 입력
→ 식재료 인식
→ 식재료 리스트 생성
→ 보유 재료 기반 메뉴 추천
→ 레시피 단계별 시각화

## 폴더 구조

- docs: 프로젝트 흐름, 화면 구성, 데이터 형식 문서
- ingredient-agent: 사진 속 식재료 인식 및 리스트 생성 담당
- recipe-agent: 식재료 리스트 기반 메뉴 추천 담당
- visual-agent: 레시피 단계별 시각화 담당
- shared: 공통 입력 예시, 프롬프트 규칙, 공통 참고 자료

## 팀 작업 방식

각 팀원은 자신의 담당 폴더를 중심으로 작업한다.

- ingredient-agent 담당자: 식재료 인식 및 리스트 생성
- recipe-agent 담당자: 메뉴 추천
- visual-agent 담당자: 레시피 시각화

공통 데이터 형식은 docs/data-format.md를 따른다.
공통 프롬프트 규칙은 shared/prompt-rules.md를 따른다.
