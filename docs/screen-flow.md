# 화면 구성

## 1. 전체 화면 흐름

홈 화면
→ 식재료 사진 입력 화면
→ 인식 결과 화면
→ 레시피 추천 화면
→ 레시피 상세 화면

## 2. 홈 화면

### 목적

앱의 기능을 간단히 안내하고, 사용자가 식재료 사진을 입력할 수 있도록 한다.

### 구성 요소

- 앱 이름
- 앱 소개 문구
- 사진 촬영 버튼
- 이미지 업로드 버튼

### 예시 문구

오늘 가진 재료로 무엇을 만들까요?

## 3. 식재료 사진 입력 화면

### 목적

사용자가 식재료가 보이는 사진을 촬영하거나 업로드한다.

### 구성 요소

- 카메라 촬영 버튼
- 이미지 업로드 버튼
- 선택한 이미지 미리보기
- 분석 시작 버튼
- 촬영 안내 문구

### 안내 문구 예시

재료가 잘 보이도록 밝은 곳에서 촬영해주세요.

## 4. 인식 결과 화면

### 목적

AI가 인식한 식재료를 사용자가 확인하고 수정할 수 있도록 한다.

### 구성 요소

- 인식된 식재료 리스트
- 재료 추가 버튼
- 재료 수정 기능
- 재료 삭제 버튼
- 메뉴 추천받기 버튼

### 예시 화면 내용

인식된 식재료

- 계란
- 대파
- 양파

버튼:

- 재료 추가
- 메뉴 추천받기

## 5. 레시피 추천 화면

### 목적

식재료 리스트를 바탕으로 만들 수 있는 한식 메뉴를 추천한다.

### 구성 요소

- 추천 메뉴 카드
- 메뉴명
- 사용되는 재료
- 조리 시간
- 난이도
- 추천 이유
- 상세 보기 버튼

### 예시 추천 카드

메뉴명: 대파계란덮밥

사용 재료:

- 계란
- 대파

조리 시간: 10분

난이도: 쉬움

추천 이유: 현재 가진 재료만으로 간단하게 만들 수 있습니다.

## 6. 레시피 상세 화면

### 목적

선택한 메뉴의 조리 과정을 단계별로 보여준다.

### 구성 요소

- 메뉴명
- 필요한 재료
- 조리 시간
- 난이도
- 단계별 조리 과정
- 간단한 요리 팁

### 예시 화면 내용

메뉴명: 대파계란덮밥

필요한 재료:

- 계란
- 대파
- 밥

조리 과정:

1. 대파를 송송 썬다.
2. 계란을 풀어둔다.
3. 팬에 대파를 볶는다.
4. 계란을 넣고 익힌다.
5. 밥 위에 올려 완성한다.
